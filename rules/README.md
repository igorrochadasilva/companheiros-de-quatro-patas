# Project rules

Este diretório define regras para os agentes (AI) que trabalham neste repositório.

- Sempre leia os arquivos de regra relevantes antes de alterar código.
- Priorize as convenções descritas em `rules/frontend.md` e nas regras de cada feature (ex.: `rules/adoption.md`).
- Nunca hardcode textos de interface: devem vir de `messages/pt-br.json`.
- Respeite a estrutura feature-based: lógica de domínio em `features/*`, UI compartilhada em `shared/ui`, utilitários em `shared/lib`.
