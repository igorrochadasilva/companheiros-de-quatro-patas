# Feature: Bazaar

Feature responsavel pela pagina publica do bazar (`/bazar`), incluindo filtros,
listagem de itens, estados de loading/erro/vazio e CTA de contato.

## Conteudo

- `services/`: consumo da API do bazar.
- `hooks/`: estado de filtros e consulta dos itens.
- `components/`: composicao da pagina e componentes de UI da feature.
